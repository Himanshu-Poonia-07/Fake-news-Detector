# Fake-news-Detector
A machine learning-based system that classifies news articles as real or fake using NLP techniques and Logistic Regression.

Fake News Detector is a project that uses computer programs to check if news articles are real or not.
It looks at the words used in the articles.
The system helps find information and promotes trustworthy information.
Here are the details of how it works:
* It uses a way called TF-IDF to turn words into numbers that the computer can understand.
* Then it uses a type of math model called Logistic Regression to make a prediction.
The Fake News Detector project is helpful for identifying misinformation and promoting information with Natural Language Processing.
The Fake News Detector project applies Logistic Regression for prediction, with TF-IDF vectorization.
