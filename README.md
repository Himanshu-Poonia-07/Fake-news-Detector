# Fake-news-Detector
A machine learning-based system that classifies news articles as real or fake using NLP techniques and Logistic Regression.

# How It Works
1. Loads and combines fake and real news datasets  
2. Cleans and preprocesses text data  
3. Converts text into numerical format using TF-IDF  
4. Trains a Logistic Regression model  
5. Predicts whether the news is fake or real  

# Results
- Achieves approximately **90–95% accuracy**  
- Fast and efficient for basic fake news detection  

# Limitations
- Accuracy depends on dataset quality  
- May not detect sarcasm or complex context  
- Needs regular updates with new data  

# Future Improvements
- Add Deep Learning models (LSTM, BERT)
